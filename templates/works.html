How We Did It

Labelling Emotions

For this part, we mined and built a collection of random images through the Imgur API. These images were shown to different people, and we used a Python script with OpenCV to capture a subject's facial reaction in response to seeing the image. By using Microsoft Project Oxford Emotion API, we recorded the emotions for each face in the captured image.

Training with Microsoft Azure

Using the dataset we generated, we used a combination of Project Oxford Computer Vision APIs and OpenCV's ORB algorithm to extract features out of the random images we had people look at. Then, we used Microsoft Azure Machine Learning to train a One vs. All multiclass classifier that utilized a Two-Class Linear Support Vector Machine. We chose a Linear SVM over Decision Trees/Forests/Jungles because of the high-dimensional feature space generated by the computer vision algorithms.

Mining Regional Data from Twitter

We utiized the TwitterSearch API to mine images tweeted by users in each of the 50 states. All of these images were then processed with the same feature extraction algorithms that were used in training our classifier.

Classifying and Displaying Data

With the features extracted from the images mined from Twitter, we used our classifier to determine the emotion reflected by the image. Aggregating the images of each state, we determine the emotional state of each region and visualize it on a map.
